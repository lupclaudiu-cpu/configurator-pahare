# Configurator Pahare Personalizate

Configurator pentru pahare albe personalizate, cu selectie pentru mai multe modele, cantitati minime, previzualizare logo si cerere de oferta pe WhatsApp.

## Rulare locala

```bash
npm install
npm run build
```

Build-ul static este generat in folderul `out`.

## Publicare Netlify Drop

Pentru publicarea rapida fara build, foloseste arhiva:

`netlify-single-file.zip`

Aceasta contine varianta standalone din `public-site/index.html`.
